# US Medicare Inpatient Cost & Payment Analysis (FY2016)

## Project Overview
This project analyzes US Medicare inpatient hospital charges and payments to understand cost variation, reimbursement efficiency, and provider-level performance.

## Tools Used
- Python (Pandas, NumPy) – Data cleaning & feature engineering
- SQL (MySQL) – Business queries and aggregations
- Power BI – Interactive dashboards and insights

## Key Analysis Areas
- High-cost DRGs and procedure-level payment gaps
- Hospitals with the highest Medicare payment gaps
- State-level variation in hospital charges and payment ratios

## Key Metrics
- Average Covered Charges
- Average Medicare Payments
- Total Payment Gap
- Medicare Payment Ratio

## Deliverables
- Cleaned dataset
- SQL business queries
- Power BI dashboard
- Executive presentation (PPT)

## Business Impact
This analysis helps identify cost drivers, reimbursement inefficiencies, and regional variation to support healthcare financial decision-making.
